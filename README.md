
# Safe Sleep

Realizar a visualização dos dados colhidos pelo sistema de monitoramento de fadiga.

## Rodando localmente

Clone o projeto

```bash
  git clone 
```

Entre no diretório do projeto

```bash
  cd your-path/safe/sleep
```

Instale as dependências

```bash
  flutter pub dev
```

Inicie a Aplicação

```bash
  flutter run lib/main.dart
```
